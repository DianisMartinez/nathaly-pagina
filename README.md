# Portfolio Profesional - Nathaly Gómez (Marketing & Presencia Digital)

Este repositorio contiene el código fuente de la plataforma web y portfolio profesional diseñado y desarrollado a medida para **Nathaly Gómez**, Periodista con mención en Comunicación Digital y Experta en Marketing Estratégico.

## 🔗 Enlace del Proyecto en Producción
La plataforma se encuentra completamente desplegada y funcional en internet a través del siguiente enlace:  
👉 **[nathaly-pagina.vercel.app](https://nathaly-pagina.vercel.app/)**

## 🎯 Objetivo del Proyecto
El sitio web fue estructurado como un portfolio estratégico de alto impacto, diseñado específicamente para presentar la trayectoria, narrativa periodística y casos de éxito de la clienta ante marcas, fundadores (*founders*) y equipos de marketing que buscan desarrollo de contenido, estrategias de UGC (User Generated Content) y posicionamiento de marca personal.

## 🚀 Características Principales del Desarrollo
* **Diseño UI/UX Premium & Minimalista:** Paleta de colores y tipografías optimizadas para transmitir autoridad, claridad y estética profesional en el sector del marketing digital.
* **Arquitectura Adaptable (Responsive Design):** Interfaz fluida y optimizada para una visualización perfecta en cualquier dispositivo (móviles, tablets y pantallas de escritorio).
* **Despliegue Continuo (CI/CD):** Integración directa con la infraestructura de Vercel para garantizar una carga ultra rápida de componentes e imágenes en alta resolución.

## 🛠️ Tecnologías Utilizadas
* **Frontend:** HTML5 semántico, CSS3 avanzado (Layouts modernos y diseño limpio) y JavaScript.
* **Despliegue y Hosting:** Vercel Cloud Infrastructure.
* **Control de Versiones:** Git & GitHub Workflow.
